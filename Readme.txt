== Description ==
Through this plugin, user can able to show the post revision done on the each post with latest update reason.

* User can able to update the latest reason for update the post.
* User can able to change the background and font color for the sections.
* Easy to add the custom text for Post Revision title heading
* Show the post revision order Ascending or Descending

For More Information : http://buffercode.com/post-revision-wordpress-plugin/

== Installation ==
1. Upload the "post-revision" directory to the plugins directory.
2. Go to the plugins setting page and activate "Post Revision". 
3. Go to Settings --> Post Revision --> Fill the Custom heading, Asc or Des order and then select the font and background color
4. Users can able to view "Reason for Last Update" in Posting area and update it.

For More Information : http://buffercode.com/post-revision-wordpress-plugin/
